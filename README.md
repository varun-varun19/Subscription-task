# Subscription-task
This project is a Subscription Task interface created using HTML and CSS, utilizing Flexbox for layout. It showcases how Flexbox can be used to design a clean and structured subscription form or component.

Features
Flexbox Layout: Efficiently aligns and distributes elements for a professional and organized appearance.
Structured Design: Includes fields and buttons typically required for a subscription form or task.
Custom Styling: Styled using CSS to enhance visual appeal while maintaining simplicity.
Static Layout: Designed with a fixed layout (not responsive).
Technologies Used
HTML5: Provides the structure for the subscription task.
CSS3: Implements Flexbox for layout and styling.
